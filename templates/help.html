RNEQC Tool
  • Release-note Quality Check (RNEQC) tool is a framework that facilitates the review and update of bug Release-note.


  • Allow customers to search for bugs using Bug ID or list of bugID's or scrubber id e.g. osman-50 or CSCvw54014 or CSCvw54014 ,CSCvw54016




  • Bug count and Filters are provided on top for filtering data present on page. Filters values get updated based on selected fields value.

  • On Page load,Offenders gets highlighted which is a bug with "Released-code"="Y" and "$$IGNORE" or "$$PREFCS" flag.

  • User can add a review comment in Add Note. Logs with user and timestamp gets captured for comments added for bug.

  • RNE status can be added and updated for a bug. Adding comments is mandatory with RNE status update. At a time only one RNE status can be attached to bug


  • RNEQC status Workflow.


  • User can ADD RNE and EDIT RNE/Release-note for bug using ADD or Edit RNE window.

  • If Release-note is not present for a bug ,missing RNE will be displayed in screen with a link to 'ADD RNE'.

  • If Release-note is present for a bug ,RNE will be displayed with better view with with all RNE contents in screen and a link to 'Edit RNE'.


  • In RNE window, Guidelines are provided for each field with all details such as 'what to consider', 'what to avoid' and 'link to Great RNE example' to write a Great RNE.

  • By default Guidelines are shown but user can close the guidelines for making more space for editing or adding RNE Contents

  • User can edit headline and other RNE fields and click on save for updating. Reset is also Provided incase user wants to undo the changes.

  • The RNE window can be closed either by clicking on dismiss button provide on top or clicking on cancel btton provided in bottom.
